# Sales Predictor AI

An intelligent sales forecasting system that leverages machine learning to predict retail sales and delivers actionable business insights powered by AI.

## Features

- Interactive Dashboard - Real-time visualization and analysis using Streamlit
- ML-Powered Predictions - Accurate sales forecasting with trained models
- AI Insights - Natural language explanations via Google Gemini AI
- Batch Processing - Upload and predict multiple records at once

## Tech Stack

- Frontend: Streamlit
- ML Framework: Scikit-learn
- Data Analysis: Pandas, NumPy
- Visualization: Plotly
- AI Engine: Google Gemini API
